### Analysis and Plotting of Differential Scanning Calorimetry (DSC), Thermogravimetric Analysis (TGA), and Differential Thermal Analysis (DTA) Plots of Commercial Polyurethane Foam

* * *
*Code written by Francis Sanguyo*

*Start Date: February 2, 2026*
*End Date: February 7, 2026*

*Description:*
This program plots and analyzes instrument data from Shimadzu DSC-60 Series and Shimadzu DTG-60A Series regarding the thermal properties of commercial polyurethane (PU) foam samples. These plots include DSC, TGA, and DTA, respectively. This program was written as part of a report for Experiment 3 of MATSE 121.02 LAB Polymeric Materials.
